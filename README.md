## Matplotlib Example: Line Plot

      import matplotlib.pyplot as plt

      y = [10, 20, 30, 40, 50]  

      plt.plot(x, y, label='Line Plot')  
      plt.title('Line Plot Example')  
      plt.xlabel('X-axis')  
      plt.ylabel('Y-axis')  
      plt.legend()  
      plt.show()
